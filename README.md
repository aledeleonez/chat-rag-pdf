# AI Assitant with Claude Sonnet 4.6 & LangChain
This project is a RAG assistant that allows you to chat with any PDF document. It is based in the Claude 4.6 Sonnet model & the LangChain framework to give precise answers based in the PDF file content.

# Tech Stack Used
- LLM: Anthropic Claude Sonnet 4.6
- Orchestrator: LangChain
- Vector DB: FAISS
- Frontend: Streamlit
- Embeddings: Sentence-Transformers (HuggingFace)
